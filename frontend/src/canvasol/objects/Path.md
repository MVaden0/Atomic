/**
 * M: move to
 * L: line to
 * C: cubic bezier
 * Q: quadratic bezier
 * A: elliptical arc
 * Z: close path
 */


# Move Commands
M: move to point (x, y)
m: move to point (dx, dy)

# Line Commands
L: line to point (x, y)
l: line to point (dx, dy)
H: line horizontal to point (x, y0)
h: line horizontal to point (dx, y0)
V: line vertical to point (x0, y)
v: line vertical to point (x0, dy)
